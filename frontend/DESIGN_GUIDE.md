## 1. The Color Palette (Change the Vibe)
All colors are stored in **Variables** at the top of `styles.css`. This is the modern way to style.
**File**: `frontend/styles.css`
**Lines**: 4-10 (`:root`)

*   **`--primary-color`**: Currently Orange (`#FF6B35`). Change this to change buttons and accents.
*   **`--bg-dark`**: Currently Black (`#0F0F0F`). Change this to `#FFFFFF` if you want Light Mode.

## 2. The Hero Image (The Big Banner)
The big image at the top is defined in CSS using a URL.
**File**: `frontend/styles.css`
**Search for**: `header` (Around Line 35)

```css
header {
    /* Change the URL inside the quotes to any image link you want */
    background: linear-gradient(rgba(0,0,0,0.7), rgba(0,0,0,0.9)), url('YOUR_IMAGE_LINK_HERE');
}
```

## 3. The Food Images (The Cards)
These are currently "Random" images generated by JavaScript.
**File**: `frontend/app.js`
**Search for**: `createKitchenCard` (Around Line 38)

To use your own specific images, you would usually save them in an `images/` folder, but for now, you can find cool food images on **Unsplash.com**, right-click "Copy Image Address", and paste them here.

## 2026 Design Trends to Try üöÄ

### A. "Bento Grids"
Instead of a simple list, make some cards big and some small, like a Bento Box.
*   **How**: In `styles.css`, look for `#kitchen-list`. Change `grid-template-columns` to experiment.

### B. "Glassmorphism" (We have this!)
The cards look like frosted glass. You can make it stronger.
*   **How**: In `.kitchen-card`, add `backdrop-filter: blur(10px);`.

### C. "Neumorphism" (Soft 3D)
Making buttons look like they are physically pressed into the screen.
*   **How**: Requires playing with `box-shadow` using both light and dark shadows.

## 4. The HTML Structure (Changing Text & Layout)
The HTML file is the "Skeleton" of your page.
**File**: `frontend/index.html`

### A. Changing the Big Title
Look for the `header` section (Around Line 16):
```html

<p class="subtitle">Experience the warmth...</p>
```
*   **Action**: Change "Cloud Kitchen" to your own brand name (e.g., "Rahul's Eats" or "Desi Delights").
*   **Action**: Change the subtitle to your own slogan.
